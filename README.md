# Basic-Login-Page
This repository contains a basic login page that one can create using the mentioned techstack in the technology section below:-
# Technology
The technology we can use are:

* HTML
* CSS
* JAVASCRIPT
* BOOTSTRAP
* NODE JS
* DJANGO <br>
<i>And many more</i>

# Tasks
* Require the username and password on the login page.

* Require full name, email address, username and a password on the sign up page.<br>
